# MisDatos VB.NET SDK

SDK en Visual Basic .NET puro (compatible con .NET Framework 4.0 y Visual Studio 2010) para consumir la API de MisDatos.

Este repositorio contiene la versión legacy diseñada para sistemas antiguos que requieran integración COM o compilación nativa en entornos .NET 4.0.

## Características
* **Compatibilidad Legacy:** Escrito en sintaxis compatible con VS 2010.
* **Sin dependencias externas:** Utiliza `System.Web.Script.Serialization` nativo del framework 4.0.
* **Compatibilidad COM:** Utiliza etiquetas `<ComVisible(True)>` para ser registrado como objeto COM en Windows de 32 o 64 bits.
